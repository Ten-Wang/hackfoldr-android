# Hackfoldr Android 快速上手

Hackfoldr 是一個收集網址的懶人包集，此為 Android 版的使用說明

加入新的頁面

在 hackfoldr page 頁面點選右上角的圖示

在中間輸入新的 key 值，或是完整網址 Ex: https://hackfoldr.org/<your_key>
